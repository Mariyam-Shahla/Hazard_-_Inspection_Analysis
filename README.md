# Hazard & Inspection Analysis

## Overview
This project focuses on analyzing hazard and inspection data using Tableau to identify inspection trends, risk patterns, and safety insights through interactive visual dashboards.

## Objectives
- Analyze hazard and inspection records
- Identify high-risk areas and inspection trends
- Monitor safety compliance patterns
- Create an interactive Tableau dashboard

## Tools & Technologies Used
- Tableau
- Data Visualization

## Dataset Information
The dataset contains hazard reports, inspection details, inspection outcomes, risk categories, and related safety information used for analytical purposes.

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Preparation
4. Data Visualization
5. Dashboard Development
6. Insight Generation

## Key Analysis Performed
- Hazard frequency analysis
- Inspection status analysis
- Risk category evaluation
- Area-wise inspection trends
- Safety compliance analysis

## Key Insights
- Certain locations reported higher hazard frequencies
- Inspection failures were concentrated in specific risk categories
- Safety compliance levels varied across different inspection areas

## Dashboard Features
- Interactive filters and visualizations
- Risk category analysis
- Inspection trend charts
- Hazard distribution visuals
- KPI and summary insights
